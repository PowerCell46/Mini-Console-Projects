    • Sofia Uni Satellite Explorer

    Sofia Uni Satellite Explorer is a project developed by Peter Gerdzhikov as part of coursework for the specialty of Application of Remote Sensing Data and Information.
    This platform offers a user-friendly interface allowing individuals to contribute satellite images enhanced with various spectral indexes. The main page serves as a dynamic showcase, providing users with a visual overview of the added satellite imagery.

    Key Features:
    1. User Contribution: The platform allows users to contribute satellite images enriched with diverse spectral indexes, fostering collaborative exploration and analysis.
    2. Interactive Visualization: Satellite imagery is prominently displayed on the main page, offering an interactive and visually engaging experience for users to explore the Earth's surface.
    3. Detailed Analysis: Each satellite image can be examined in detail through a dedicated view, providing options to download the image and manipulate spectral indexes for in-depth analysis.
    4. Spectral Index Customization: Users have the flexibility to alter spectral indexes, enhancing the utility of the satellite imagery for a variety of applications.
